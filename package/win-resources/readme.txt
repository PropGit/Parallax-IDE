Parallax IDE
=======================

Introduction
-----------------

The Parallax IDE is a completely redesigned BASIC Stamp Editor for Parallax BASIC Stamp Microcontrollers.
This tool provides PBASIC source code editing, compiling, and downloading functions on your computer without
the need for an Internet connection.

Supports BS2, BS2e, BS2sx, BS2p24/40, and BS2pe.  Supports single program slot only.


Running
-----------------

The only requirement is that FTDI USB Serial Drivers are installed to allow serial connection over usb.
